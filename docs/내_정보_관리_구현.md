# 내 정보 관리 구현 상세

## 개요
기존에 없던 사용자의 개인 정보 관리를 위한 인터페이스를 각각의 사이드바 최상단에 배치하고 프로필 수정 기능을 제공합니다.

## 개발 범위

### 1. 사용자 접근 경로
- **학습자/교수자/관리자 공통**: 좌측 전역 사이드바 최상단에 사용자 닉네임과 함께 `[내 정보]` 메뉴 고정 배치.
- 탭 이동 형태로 구현되어 대시보드 컴포넌트의 가시 영역만 전환해 즉각적인 접근성 확보.

### 2. 화면 명세 (UI)
- 현재 계정의 ID(수정 불가) 및 현재 역할(Role) 표시
- 사용자 실명 텍스트 입력 폼
- **새 비밀번호 설정 및 실시간 검증 [UPDATE]**
  - 새 비밀번호와 비밀번호 확인 입력 필드 제공.
  - 사용자가 입력하는 즉시 두 비밀번호의 일치 여부를 하단에 실시간 텍스트로 안내하여 편의성 및 데이터 정확도 향상.

- 실명 변경 시 `users.update` API 활용을 통해 전역 사용자 데이터 스토리지 업데이트
- 비밀번호 변경 시 **현재 비밀번호 확인 과정**을 거친 후 `users.updatePassword` API를 통해 안전하게 처리
  - 제출 시 기존 비밀번호가 일치하지 않을 경우, 입력 필드 하단에 즉각적인 인라인 텍스트로 에러 메시지를 표시하여 사용자가 쉽게 인지하도록 안내
- 수정 제출 시 즉각적인 Feedback(성공 Alert) 제공 및 상단 프로필 실명 즉시 동기화
- **전역 상태 동기화 무결성**: `MyInfoModal`에서 정보 수정 후 `updateUser` 호출 시, 사용자의 `userId`와 `role` 정보가 유실되지 않도록 `AuthContext` 내부의 방어 로직을 통해 세션의 연속성을 보장합니다.
