# 영상 강의 업로드 및 재생 구현 상세

## 개요
LMS의 핵심 콘텐츠인 동영상 강의를 관리하고 재생하기 위한 전용 파이프라인(업로드, 리스팅, 유튜브 연동) 설계서입니다.

## 주요 구현 사항

### 1. 전용 업로드 파이프라인 (`UploadForm.jsx`, `/admin/class/[id]/upload/page.js`)
- **분리된 페이지 설계**: 강의 업로드는 많은 메타데이터 편집(제목, 설명, 영상 링크)이 요구되므로, 복잡한 모달창 대신 단독 페이지 라우팅 방식으로 구현.
- **미리보기 제공**: 사용자가(관리자/교수) 입력한 Youtube URL이 올바른지 사전에 점검할 수 있도록 `iframe` 기반의 **YouTube Video 미리보기 렌더링** 지원. URL Parsing 정규식을 활용해 긴 URL, 단축 URL 등 형태를 자동 추출 후 포맷팅.

### 2. 강의 리스트업 관리 (`LectureList.jsx`)
- 클래스 상세 뷰의 '강의 목록' 탭 전용 컴포넌트화 작업 진행.
- 전체 강의 항목이 번호순/등록순으로 정렬되며, 각 항목별 관리 버튼(수강률 통계 조회, 편집, 삭제)이 Grid 형태로 부착됨.

### 3. 시청 및 재생 환경 (`/lecture/[id]/page.js`)
- **독립된 뷰어 환경**: 몰입도 향상을 위해 사이드바 등의 방해요소를 최소화한 전용 독립 페이지 (`lecture-layout` CSS 적용).
- **상태 보존**: 다른 페이지를 다녀와도 이전에 보고 있던 콘텐츠 맥락으로 다시 돌아올 수 있도록 Header 메뉴에 즉시 접근 가능한 `<Back>` 버튼 배치.
- 재생 완료 시 진도율 체크 API(`api.views.updateProgress`)와 연동하여 자동으로 수강 100% 상태를 마킹하도록 설계됨.
