# 학습 진도율 및 통계 시스템 상세 구현

## 개요
개별 학습자의 학습 모니터링 및 교수/관리자의 코칭을 지원하기 위한 강의별 진도율 산출 및 통계형 뷰어 UI 구현 내용입니다.

## 주요 구현 사항

### 1. 학생 대시보드 진도율 게이지 (Progress Bar)
- **연산 처리**: 학생의 내 학습 공간 목록 렌더링 시, 각 클래스에 속한 `Total Lectures Count` 와 학생이 시청 완료 처리된 API 반환 Count를 대조.
- **UI 반영**: `(완료 수 / 전체 강의 수) * 100` 수식을 사용하여 학생 대시보드 카드 하단에 HTML5/CSS 커스텀 프로그레스 바(게이지 바) 및 백분율 퍼센티지 텍스트를 직관적으로 표시.

### 2. 관리자/교수자 권한 - 개별 클래스/강좌 수강 통계 뷰 (`/lecture_stats/[id]/page.js`)
- **접근**: 클래스의 '강의 목록' 요소 중 특정 챕터의 **[수강률 통계]** 버튼 클릭 시 전용 페이지로 이동.
- **종합 조회**: 이 강의가 포함된 클래스의 전체 수강생 목록과 대조하여 어떤 학생이 본 챕터를 얼마나 시청했는지 시각적 표(Table)로 반환.
- **성취도 색상**: 진도율 95% 이상 시 '수강 완료' 상태로 표시하되, 실제 시청한 비율(95%~100%)을 배지에 함께 노출하여 정밀한 상태 모니터링 지원.
- **수강 상태 배지 (`StatusBadge`)**: `95% 이상: 수강 완료`, `1%~94%: 수강 중`, `0%: 미수강`으로 구분하여 시각화.

### 3. 학생 개별 활동 포트폴리오 (`StudentDetail.jsx`)
- **접근**: 수강생 관리 명단에서 특정 학생의 **[상세 보기]** 클릭.
- 특정 학생 1명이 해당 클래스 안에서 보여준 모든 이력을 단일 페이지에서 집계:
  - 그동안 시청한 강의들의 진도율 표
  - 해당 클래스 내부에서 작성/질문한 QnA 활동 내역 타임라인 연동
- 개별 피드백 및 학점 산출 등의 증빙 자료로 활용 가능하도록 구현.
