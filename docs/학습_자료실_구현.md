# 학습 자료실 시스템 상세 구현

## 개요
강의 영상 외에 필요한 보조 파일이나 외부 링크 문서를 등록, 확인, 다운로드할 수 있도록 구성한 클래스별 부가 컴포넌트 기능입니다.

## 주요 구현 사항

### 1. 운영 UI 구조 (`ResourceList.jsx`, `ResourceForm.jsx`)
- **조회**: 클래스 상세 보기 뷰 내부에서 '자료실' 탭 클릭 시 노출. 교수/학생 공통으로 조회되는 리스트 형식의 전용 목록.
- **업로드**: `ResourceForm.jsx` 모달 창을 통해 담당 교수자 혹은 관리자만 신규 자료 레코드를 생성할 수 있음. 학생은 오직 Read 만 허용된 모델.

### 2. 데이터 속성 관리
- **저장 항목**: 
  - `title` (자료 제목 - 예: "Week 01. 자바 강의 교안")
  - `description` (자료에 대한 상세 설명 및 참고 사항)
  - `filename` (첨부파일의 이름 또는 다운로드 링크 대용. 초기 버전에서는 스트링 기반 다운로드 명세를 흉내 냄)
  - `lectureId` (특정 강의 회차와 연동될 경우 사용할 파라미터. 현재는 전역으로 노출)

### 3. 관리자 권한 연동 및 로깅
- 교수 및 관리자 계정으로 조회할 때만 각 자료 리스트 우측에 `[수정]`, `[삭제]` 제어 버튼이 렌더링 됨. 학생 화면에서는 숨김 처리.
- 자료 편집, 삭제, 추가 시 역시 중앙화된 `api.logs` 이벤트 캐치 모듈에 전송되어 해당 클래스 전체 타임라인 내역(`ActivityLogModal`)에 "첨부 파일 등록/삭제" 문구로 기록 갱신됨.
