# 학습 자료실 시스템 상세 구현

## 개요
강의 영상 외에 필요한 보조 파일이나 외부 링크 문서를 등록, 확인, 다운로드할 수 있도록 구성한 클래스별 부가 컴포넌트 기능입니다.

## 주요 구현 사항

### 1. 운영 UI 구조 (`ResourceList.jsx`, `ResourceForm.jsx`)
- **조회**: 클래스 상세 보기 뷰 내부에서 '자료실' 탭 클릭 시 노출 및 단일 강의 재생 화면 우측 사이드바 노출.
- **업로드**: `ResourceForm.jsx` 모달 창을 통해 담당 교수자 혹은 관리자만 신규 자료 레코드를 생성할 수 있음. 제목 `title` 입력란은 미입력 시 시스템이 원본 파일명을 기본 제목으로 자동 지정함. `FormData` 파싱을 통해 `File` 객체 원본을 서버로 발송.
- **다운로드**: 관리자, 교수자, 학생 모두 브라우저 `window.URL.createObjectURL` 기반의 실물 Blob 강제 다운로드 버튼 사용 가능 (`api.resources.download`).

### 2. 데이터 속성 관리
- **저장 항목**: 
  - `title` (자료 제목 - 예: "Week 01. 자바 강의 교안". 자동/수동 지정)
  - `description` (자료에 대한 상세 설명 및 참고 사항)
  - `filename` (첨부파일의 원래 이름 식별자)
  - `lectureId` (특정 강의 회차와 연동될 경우 사용할 파라미터. 현재는 전역으로 노출)

### 3. 관리자 권한 연동 및 로깅
- 교수 및 관리자 계정으로 조회할 때 자료 리스트 우측에 `[삭제]` 제어 버튼이 렌더링 됨. 학생 화면에서는 파일 다운로드 버튼만 노출됨(학생 외 교수/어드민도 다운로드는 제약 없이 사용).
- 자료 편집, 삭제, 추가 시 역시 중앙화된 `api.logs` 이벤트 캐치 모듈에 전송되어 해당 클래스 전체 타임라인 내역(`ActivityLogModal`)에 "첨부 파일 등록/삭제" 문구로 기록 갱신됨.
